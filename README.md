# Claude Code za 90 dana / in 90 Days / in 90 Tagen

Besplatan kurs samostalnog učenja [Claude Code-a](https://code.claude.com/docs) — od prve komande do sopstvenih agenata i automatizacije. 80% prakse, 20% teorije.

*A free self-study course for Claude Code — available in Serbian, English and German. / Ein kostenloser Selbstlernkurs für Claude Code — auf Serbisch, Englisch und Deutsch.*

## Šta je u kursu

- **6 modula / 12 nedelja:** Claude CLI osnove, CLAUDE.md i memorija, Plan mode i Loop, MCP serveri, paralelni subagenti, Git integracija i automatizacija
- **6 mini-projekata od nule** — ne treba ti nikakav postojeći kod, samo instaliran Claude Code i osnove terminala
- **Rešenja svih zadataka** na kraju kursa: referentni promptovi, česte greške i kriterijumi uspeha
- Umesto sertifikata: završni projekat sa checklist-om koji pokriva sve naučeno

## What's in the course

- **6 modules / 12 weeks:** Claude CLI basics, CLAUDE.md and memory, Plan mode and Loop, MCP servers, parallel subagents, Git integration and automation
- **6 mini-projects from scratch** — no existing code needed, just Claude Code installed and terminal basics
- **Solutions to all exercises** at the end: reference prompts, common mistakes and success criteria
- Instead of a certificate: a final project with a checklist covering everything learned
